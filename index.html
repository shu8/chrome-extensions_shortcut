Chrome://extensions hotkey

Press Ctrl+Shift+E (Cmd+Shift+E on OSX) to open the chrome://extensions page

Download this project as a .zip file Download this project as a tar.gz file

To use / configure:

  • Ctrl+Shift+E to open chrome://extensions.
  • Go to chrome://extensions/configureCommands and find chrome://extensions shortcut. Type your preferred shortcut.

Why would I need this?

I don't know about you, but I started developing Chrome Extensions a few weeks ago, and opening http://chrome-extensions every time is already a pain. I don't want to keep that tab open, because I have too many tabs open already.

I'm new to Chrome Development so, to help me learn about chrome.commands, I created this extension!
